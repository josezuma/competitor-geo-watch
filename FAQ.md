# FAQ

## What is competitor-geo-watch?
Agent skill that watches competitor GEO scores and alerts on changes.

## How do I install it?
```bash
git clone https://github.com/josezuma/competitor-geo-watch.git
cd competitor-geo-watch
python3 scripts/generate.py --help
```

## Do I need any external dependencies?
No. All scripts use Python stdlib only.
